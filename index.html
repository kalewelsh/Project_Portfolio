Mountain Project Sport Climbing Route Analysis

Overview I wrote a Python script to scrape data such as rout length, user quality rating, difficulty rating, and more from all of the sport climbing routs on the MountainProject website. I created an interactive map to how sport climbing routes differ between the states. I created a density map to get an idea of the parts of the U.S that have the most sport climbing routes and to provide a more granular view of where rock climbing routes are located than what is offered on the MontainProject website.

Car Accident Clustering Tool

Link To Car Accident Clustering Web Application: https://share.streamlit.io/kalewelsh/car-accident-clustering-tool/main/Car-Accident-Clustering.py Link to Github Repo: https://github.com/kalewelsh/Car-Accident-Clustering-Tool Overview I created a web application that can be used to identify clusters of car accidents using a density clustering algorithm. I created this application using the Python packages Streamlit, Pandas, Numpy, ScikitLearn, and Plotly. I designed the web application in a way that allows the user to modify the parameters of the algorithm, changing the way that the algorithm clusters the data.
